"오늘의 길"(이하 "회사")은 이용자의 개인정보를 소중하게 생각하며, 「개인정보 보호법」, GDPR(EU 일반 개인정보 보호법), CCPA(캘리포니아 소비자 개인정보 보호법) 등 관련 법령을 준수하고 있습니다. 본 개인정보처리방침은 이용자가 앱을 사용할 때 어떠한 정보가 수집·사용·저장되는지를 안내합니다.

## 1. 개인정보 처리 원칙 — 로컬 우선(Local-First)

"오늘의 길"은 **모든 이용자 데이터를 이용자 기기(로컬)에만 저장**하는 것을 기본 원칙으로 합니다.

- 회사는 이용자의 위치, 경로, 감정 기록, 사진, 메모 등 어떠한 개인정보도 **서버에 수집·전송·저장하지 않습니다.**
- 회사는 이용자 **계정 시스템을 운영하지 않으며**, 회원가입 없이 앱을 사용할 수 있습니다.
- 이용자의 데이터는 앱을 삭제하는 즉시 기기에서 영구적으로 삭제됩니다.

## 2. 수집 및 이용 항목

"오늘의 길"은 아래와 같은 정보를 **이용자 기기 내에서만** 처리합니다.

| 항목                            | 수집/처리 방법         | 이용 목적                        | 저장 위치      | 보관 기간          |
| ------------------------------- | ---------------------- | -------------------------------- | -------------- | ------------------ |
| 위치 정보 (GPS 좌표, 이동 경로) | OS 위치 서비스         | 산책/등산 경로 기록, 지도 표시   | 기기 내 SQLite | 이용자 삭제 시까지 |
| 걸음 수                         | OS 활동 감지 센서      | 운동량 기록                      | 기기 내 SQLite | 이용자 삭제 시까지 |
| 사진                            | 카메라/사진 라이브러리 | 산책 기록에 사진 첨부            | 기기 저장소    | 이용자 삭제 시까지 |
| 감정/무드, 메모                 | 이용자 직접 입력       | 감정 기반 산책 일기 작성         | 기기 내 SQLite | 이용자 삭제 시까지 |
| 알림 상호작용                   | OS 알림 시스템         | 산책 리마인더, 진행 중 알림 표시 | 기기 내        | 앱 실행 중         |

> 본 앱은 이용자의 실명, 주민등록번호, 이메일, 전화번호 등 **식별 가능한 개인정보를 일체 수집하지 않습니다.**

## 3. 앱 권한 안내

원활한 서비스 제공을 위해 아래 권한을 요청하며, 이용자는 언제든지 OS 설정에서 권한을 회수할 수 있습니다.

### 필수 권한

- **위치 정보(FINE_LOCATION):** 산책 경로 기록을 위한 정확한 GPS 좌표 획득
- **활동 인식(ACTIVITY_RECOGNITION / Motion):** 걸음 수 측정

### 선택 권한

- **백그라운드 위치(BACKGROUND_LOCATION):** 화면이 꺼진 상태에서도 산책 경로를 끊김 없이 기록 (등산·장거리 산책용)
- **알림(POST_NOTIFICATIONS):** 산책 시작/종료, 리마인더 알림
- **정확한 알람(SCHEDULE_EXACT_ALARM):** 지정 시간 리마인더
- **카메라/사진 라이브러리:** 산책 기록에 사진 첨부
- **Foreground Service (LOCATION):** 산책 진행 중 OS에 의해 앱이 중단되지 않도록 지속적 추적 보장

권한을 거부하시더라도 대부분의 기능을 이용할 수 있으며, 해당 권한이 필요한 기능만 제한됩니다.

## 4. 제3자 제공 및 처리 위탁

회사는 이용자의 개인정보를 **제3자에게 제공하거나 판매하지 않습니다.** 다만 서비스 운영에 필요한 아래 항목은 예외적으로 처리 위탁됩니다.

### 4-1. 제3자 서비스

| 제공자                       | 처리 항목                    | 처리 목적       | 전송 방식      |
| ---------------------------- | ---------------------------- | --------------- | -------------- |
| Google LLC (Google Maps)     | 익명 지도 타일 요청 로그     | 지도 렌더링     | 요청 시점에만  |
| Apple Inc. (Apple Maps, iOS) | 익명 지도 타일 요청 로그     | iOS 지도 렌더링 | 요청 시점에만  |

| Sentry (Functional Software Inc.) | 앱 에러/크래시 로그 (비식별·익명) | 앱 안정성 개선 및 오류 모니터링 | 에러 발생 시 |

- Firebase Crashlytics 및 Sentry는 기기 모델, OS 버전, 앱 버전, 스택트레이스 등 **기술 정보만** 수집하며, 이용자는 앱 설정에서 크래시 리포팅을 비활성화할 수 있습니다.
- 각 제3자 서비스의 개인정보 처리 방침:

- 미국 COPPA에 따라 만 13세 미만 아동의 개인정보 수집 대상이 아닙니다.

## 7. 이용자의 권리 (GDPR / CCPA 포함)

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

### 기본 권리

- **열람권:** 앱 내 기록 화면에서 모든 데이터를 직접 확인
- **수정권:** 기록된 일기·메모·사진을 앱 내에서 직접 수정
- **삭제권 ("잊혀질 권리"):** 개별 기록 삭제 또는 앱 삭제를 통한 전체 데이터 즉시 삭제
- **처리 제한권:** OS 설정에서 권한 회수를 통한 처리 중단
- **이동권(Portability):** 향후 데이터 내보내기 기능 제공 예정 (JSON/CSV)

### GDPR 권리 (EU/EEA 거주자)

- 회사는 이용자 데이터를 서버에 저장하지 않으므로, 별도의 "데이터 제공·삭제 요청"을 접수할 필요가 없습니다.
- 다만 문의 사항이 있을 경우 아래 연락처로 요청하시면 30일 이내에 답변드립니다.
- 감독기관 이의제기권: EU/EEA 거주자는 관할 데이터 보호 감독기관에 이의를 제기할 수 있습니다.

### CCPA 권리 (캘리포니아 주민)

- 회사는 이용자의 개인정보를 **판매(sale)하지 않습니다.** ("Do Not Sell or Share My Personal Information")
- 이용자는 차별 없이 본 개인정보처리방침의 권리를 행사할 수 있습니다.

## 8. 개인정보의 국외 이전

- 회사는 이용자의 개인정보를 국외로 이전하지 않습니다.
- 다만 제3자 서비스(Google Maps, Firebase 등)는 각 제공자의 정책에 따라 글로벌 인프라에서 처리될 수 있으며, 이는 각 서비스의 개인정보 처리방침을 따릅니다.

## 9. 개인정보의 안전성 확보 조치

회사는 로컬 우선 원칙에 따라 데이터가 이용자 기기에만 저장되도록 설계하였으며, 아래 조치를 시행합니다.

- 기기 내 데이터는 OS 수준의 앱 샌드박싱으로 격리
- SQLite 데이터베이스는 앱 전용 저장소에 저장되어 다른 앱에서 접근 불가
- 외부 네트워크 전송 없음 (지도 타일, 크래시 리포팅 제외)
- 앱 업데이트 시 마이그레이션 스크립트 검증
